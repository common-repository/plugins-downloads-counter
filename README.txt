== Description ==

Get combined download counts for all your plugins on WordPress.org.

The plugin uses the http://wptally.com/ API.

== Installation ==

1. Upload the entire plugins-downloads-counter folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Use shortcode `[plugins_downloads_counter username="yourusername"]` or add 'Plugins Downloads Counter' widget in any post/page content to display your plugins' download counts.

== Frequently Asked Questions ==

= Which parameters can I pass to the shortcode? =

You can pass the username parameter:

`[plugins_downloads_counter username="yourusername"]`
